Passion Or Paycheck?

Welcome!

Passion Or Paycheck is a web system for the integrity of schools' devotion to the next generation's education.

We do this to show students about their schools and if their school's staff put their heart truly towards the next generation instead of profit.

Passion Or Paycheck is a non-profit, open-source website as a side project for GitHub user ojg1. If you would like information removed, please email ericlin1253@gmail.com
